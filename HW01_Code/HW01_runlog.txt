notes:
h_misplaced:
  - Admissible: Never overestimates (each misplaced tile needs ≥1 move)
  - Consistent: Moving a tile changes misplaced count by at most 1
  - Weak heuristic: Underestimates significantly, explores many nodes

h_manhattan:
  - Admissible: Manhattan is minimum moves ignoring obstacles
  - Consistent: Moving a tile changes its Manhattan distance by ±1
  - Dominates h_misplaced: h_manhattan(n) ≥ h_misplaced(n) for all n
  - Strong heuristic: Better estimates → far fewer nodes expanded
  - Result: Usually finds solution faster despite more complex computation
